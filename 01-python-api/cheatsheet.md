# Platform Repo Cheatsheet

## Setup

```bash
export GITHUB_OWNER="your-github-org"
export GITHUB_TOKEN="ghp_YOUR_TOKEN_HERE"
cd 01-python-api
```

* my_account/repo-a-public

Create one team in your organization:
